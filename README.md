# 🧮 Tugas Bahasa Pemrograman 2 Matrix 5x5 

Skrip Python sederhana untuk membuat matriks 5x5 menggunakan metode `append()`.

Setiap elemen diisi otomatis dari angka 1 sampai 25, membentuk matriks dua dimensi.
